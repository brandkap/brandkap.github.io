ConnectOurKids

Forever families found faster

Background

Connectourkids is a non-profit organization that aims to connect foster children with a permanent home. Their organization provides free tools to social workers to help expedite their work. Our goal was to take the main features of the ConnectOurKids website and create a mobile app incorporating those features. We had weekly meetings with the CTO of ConnectOurKids.

Role

UX Design

Duration

Dec 2019 - Feb 2020

The problem: Lack of mobile tools

ConnectOurKids provides free tools to social workers, but the dilemna is that these tools can only be accessed when in front of a desktop computer.The limitation of tools available to social workers in a mobile setting makes the process of finding permanent homes for foster children more delayed than necessary.There is a gap in the work flow of a social worker from the lack of having mobile tools.

Providing value to social workers

Before designing any screens it was important to define the key features and understand how social workers would get value out of the app. Through discussions with the CTO and an understanding of the work flow of social workers based off the wesbite we were able to layout the most important features.

Creating a mobile experience

Our reference for designing the mobile screens was the ConnectOurKids website. We reviewed the existing website designs, brainstormed ideas, and took notes of things we wanted to incorporate in the future mobile designs. Our goal was to create a mobile experience that maintained the same functionality and organization as the website. We also made sure to keep an eye out for opportunites to optimize and improve the experience. The two main features we worked on were People Search and Family Connections.

1. People Search

Simply put, people search is the way social workers find who they are looking for in the world. This search function searches through public databases containing over 3 billion people. Potential candidates for a foster child can be located through here.

Desktop version of People Search

Some notes from reviewing the website designs:

  • There should be more a consistent color scheme; there are too many shades of blue
  • Clearer indication of what searching criteria is selected
  • Text input fields should be separated
  • Drop down menu selectors for city and state would save the headache of having to type out the entire city/state

Results from a people search

Search results are displayed in a pop up that allows for a quick return back to the search home.

2. Family Connections

All of potential candidates for permanent housing for a foster child are housed in the Family Connections section. Social workers can log engagement, edit personal information, and upload paperwork.

Desktop version of Family Connections

Some notes from reviewing the website designs:

  • On mobile we will not have enough room to display people in this grid format
  • Having a side pop up after clicking on a persons card will make the screen too cluttered. We will need an entirely separate screen decicated to the full profile of a child
  • Displaying of a childs family connections should be the first screen after clicking into a childs profile

Organizing Family Connections

A color system was created to quickly identify what type of family connection is being viewing.

Consistent design system

Our interface choices were based on the design elements already present on the website. We did not want to reinvent the wheel and have the app vary drastically. So in order to keep the experience as similar as possible between the website and mobile app we use the same using consistent: copy, color schemes, typeface, and navigation layout.

Validating our designs

Main goals of user testing:

  • Identify any usability issues regarding searching, filtering, and editing contact information
  • Also, seeing if the participants familiarity with the ConnectOurKids website transferred into easy use of the app.
  • get personal opinions and feedback on if these features would would actually help them in real life.
